✨ 용사 키우기에 오신 선생님, 환영합니다! ✨

선생님의 학급을 즐거운 모험의 세계로 만들 준비가 되셨나요?
본격적인 시작에 앞서 핵심 기능 4가지를 안내해 드립니다.

1. 🏫 나만의 학급 만들기

대시보드 상단의 [+ 새 학급 추가] 버튼을 눌러 학급을 생성해 주세요. 학생들을 초대할 수 있는 6자리 고유 코드가 발급됩니다.

2. 🧑‍🤝‍🧑 학생 초대 및 관리

학생들이 회원가입 시 초대 코드를 입력하면 해당 학급으로 자동 배정됩니다. 학생 카드에서 닉네임 변경(✏️)이나 삭제(❌)가 가능합니다.

3. 💰 포인트 부여와 성장

학생들에게 칭찬 포인트를 부여해 보세요! 직업은 [기사 - 마법사 - 소환사] 세 가지가 있으며, 포인트를 모을 때마다 [알 → 초급 → 중급 → 최종] 단계로 진화합니다.
100포인트를 모을 때마다 새로운 직업으로 전직하게 됩니다.

4. ⚡ 나만의 맞춤 버튼 추가

자주 사용하는 칭찬이나 벌점이 있다면, [+ 버튼 추가] 기능을 활용해 보세요. 버튼 하나로 빠르고 간편하게 여러 학생에게 포인트를 줄 수 있습니다.
